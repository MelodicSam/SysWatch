## 🌌 Overview

**SysWatch** is a real-time system monitoring web application that provides deep insights into:

- CPU Usage (Live + Per Core)
- Memory & Swap Monitoring
- Disk Utilization
- Network Activity
- Running Processes
- System Information

All displayed in a **clean, modern, blue-themed dashboard UI**.
